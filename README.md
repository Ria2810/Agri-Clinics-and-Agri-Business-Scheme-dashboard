# Agri Clinics and Agri-Business Scheme Dashboard 🌾📊

This repository hosts an **interactive Tableau dashboard** for the Agri Clinics and Agri Business Scheme. It delivers clear insights and visualizations on key indicators like trained candidates, subsidy amounts, loans sanctioned, subsidies sanctioned, and agri-ventures established. This dashboard offers a comprehensive view of the scheme’s performance and its impact on agri-business development.

## 1. Introduction 🚜🌱

The **Agri Clinics and Agri Business Scheme** is a government initiative that promotes entrepreneurship and agri-business development in the agricultural sector. It provides training and financial support to individuals and organizations setting up agri-clinics, agribusiness centers, and agri-ventures.

This interactive dashboard, built with Tableau, presents key performance indicators of the scheme in a visually engaging format. It enables stakeholders and policymakers to monitor progress, analyze trends, and assess the scheme's impact over time.

---

## 2. Dashboard Overview 👀

The dashboard offers insights into critical indicators:

- **Trained Candidates**
- **Subsidy Amount**
- **Loans Sanctioned**
- **Subsidies Sanctioned**
- **Agri-Ventures Established**

These indicators are showcased in an easily accessible and visually appealing format, empowering users to quickly understand the scheme’s performance and trends.

## 3. Dashboard Sections 🗂️

### 1. Overview
- **Indicator Counts**: Displays the total counts for all key indicators.
- **State-Based Breakdown**: Visualizes indicator counts by state.
- **Year-on-Year Change**: Highlights changes over time.
